Login for a user

Check how user see system without password drop

The module adds button "Generate login link" to User form in debug mode.

After you press on it, system will generate an access link you can use in browser private mode to login for a choosen user.
You can use generated link only once. Life time of the link by default is 60 seconds.You can change it in controller.
